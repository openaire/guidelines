.. _d:language:

9. Language (R)
---------------
The primary language of the resource (occurrences: 0-1).

**Allowed values, examples, other constraints**

Allowed values are taken from IETF BCP 47, ISO 639-1 language codes.
Examples: ``en``, ``de``, ``fr``.

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <language>eng</language>
